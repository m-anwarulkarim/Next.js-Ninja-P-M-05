Next.js-এ **`redirect`** ফাংশনটি মূলত ইউজারকে এক পেজ থেকে অন্য পেজে পাঠিয়ে দেওয়ার জন্য ব্যবহার করা হয়। এটি অনেকটা ট্রাফিক পুলিশের মতো কাজ করে, যে ট্রাফিককে এক রাস্তা থেকে অন্য রাস্তায় ডাইরেক্ট করে দেয়।

সহজ ভাষায় এর কাজগুলো নিচে দেওয়া হলো:

### ১. কেন এটি ব্যবহার করবেন?

- **লগইন প্রোটেকশন:** যদি কোনো ইউজার লগইন না করে প্রোফাইল পেজে যেতে চায়, তবে তাকে `redirect` ব্যবহার করে লগইন পেজে পাঠিয়ে দেওয়া হয়।
- **ফর্ম সাবমিশন:** ইউজার একটি ফর্ম ফিলাপ করার পর তাকে "Thank You" পেজ বা ড্যাশবোর্ড পেজে পাঠানোর জন্য এটি ব্যবহৃত হয়।
- **পুরানো লিঙ্ক:** আপনার সাইটের কোনো পেজের লিঙ্ক পরিবর্তন করলে, পুরানো লিঙ্কে কেউ ভিজিট করলে তাকে নতুন লিঙ্কে পাঠিয়ে দিতে এটি কাজে লাগে।

---

### ২. কোড উদাহরণ (Server Component):

Next.js-এ এটি `next/navigation` থেকে ইমপোর্ট করতে হয়।

```javascript
import { redirect } from "next/navigation";

export default async function ProfilePage({ params }) {
  const user = await fetchUser();

  if (!user) {
    // ইউজার না থাকলে লগইন পেজে পাঠিয়ে দাও
    redirect("/login");
  }

  return <div>Welcome to your profile!</div>;
}
```

---

### ৩. গুরুত্বপূর্ণ কিছু পয়েন্ট:

- **Error Handling:** আপনি যখন `redirect()` কল করেন, তখন Next.js আসলে পর্দার আড়ালে একটি স্পেশাল **"Redirect Error"** থ্রো (throw) করে। এটি কোনো সাধারণ ভুল বা বাগ নয়, বরং এটি Next.js-কে জানানোর একটি উপায় যে "এই পেজটি রেন্ডার করা বন্ধ করো এবং ইউজারকে অন্য পেজে নিয়ে যাও।"
- **Try-Catch এর ভেতরে ব্যবহার:** যেহেতু এটি ইন্টারনালি একটি 'error' থ্রো করে, তাই একে `try...catch` ব্লকের ভেতরে লিখলে এটি কাজ করবে না (ক্যাচ ব্লক এটিকে সাধারণ এরর ভেবে আটকে দেবে)। তাই সবসময় এটি `try...catch`-এর বাইরে ব্যবহার করার চেষ্টা করবেন।

### ৪. ক্লায়েন্ট সাইড বনাম সার্ভার সাইড:

- **Server Component:** এখানে সরাসরি `redirect()` ব্যবহার করা হয়।
- **Client Component:** এখানে সাধারণত `useRouter()` হুক ব্যবহার করে `router.push('/path')` করা হয়, তবে Next.js ১৫+ ভার্সনে ক্লায়েন্ট সাইডেও `redirect` সাপোর্ট করে।

**একটি জরুরি টিপস:**
যদি আপনি এমন কোনো এরর ফেস করেন যেখানে বলা হচ্ছে "NEXT_REDIRECT", বুঝবেন যে `redirect()` ফাংশনটি কাজ করছে কিন্তু কোনোভাবে সেটি `try-catch` ব্লকের ভেতরে আটকে গেছে।
